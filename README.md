# Emotion Detection AI using BERT

This project is an AI-powered Emotion Detection system built using BERT and PyTorch.
It predicts emotions such as Happy, Sad, Angry, Fear, Excited, and Neutral from text.

## Features
- BERT-based emotion classification
- Command-line prediction
- Web application using Flask
- Confidence score output

## Tech Stack
- Python
- PyTorch
- HuggingFace Transformers
- Flask
- HTML & CSS
